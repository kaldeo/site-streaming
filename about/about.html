Accueil À propos Programmes Abonnements Compte

Voici quelques informations qui pourraient vous aider à mieux comprendre le site :

Accueil :

La page d'accueil est juste une page décorative où l'on peut retrouver le nom du site.

Programmes :

Sur la page programme on peut retrouver tous les programmes que l'on propose, pour pouvoir accéder à nos programmes vous êtes dans l'obligation de créer un compte et je choisir un abonnement.

Abonnements :

Je vous propose 3 abonnements différents que vous pouvez retrouver sur cette page. Pour souscrire un abonnement il suffit de choisir votre abonnement et d'aller sur “voir détails” puis acheter l'abonnement, mais vous devez créer un compte avant.

Compte :

Et pour finir sur la page compte vous avez la possibilité de vous créer un compte et de vous connecter, une fois cela fait vous pourrez retrouver vos informations de compte en direct et vous pouvez également les modifier. Si vous le souhaitez sur cette page vous pouvez également résilier l'abonnement que vous avez souscrit.
